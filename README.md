# 名古屋滑雪之旅 🐱

> 2026/02/06 – 02/14｜9 天 8 夜名古屋 & 長野自駕旅遊行程 Web App

## 簡介

這是一個以**單頁 HTML** 打造的旅行行程管理工具，專為「名古屋 + 長野滑雪家庭旅遊」設計。整個應用程式僅靠一個 `index.html` 即可運行，無需後端伺服器，所有資料以 `localStorage` 儲存於瀏覽器本地端。

## 功能模組

| 模組 | 說明 |
|------|------|
| **行程** (Plan) | 9 天逐日行程時間軸，含航班資訊、Google Maps 導航連結、地址複製 |
| **導覽** (Guide) | 景點攻略卡片（JR Gate Tower、鈴鹿賽道飯店、池之平飯店、吉卜力公園），含圖片輪播與停車場資訊 |
| **記帳** (Wallet) | 日幣消費記錄、匯率換算（JPY → TWD）、算式計算機、拍照附圖功能 |
| **清單** (Lists) | 旅遊行李 & 滑雪裝備打勾清單，支援自訂項目新增、長按刪除 |
| **資訊** (Info) | 天氣預報連結、緊急聯絡電話、華航行李規定、旅遊注意事項 |

## 技術棧

- **HTML5** — 單頁應用 (SPA)
- **Tailwind CSS** (CDN) — 響應式 UI 樣式
- **Font Awesome 6** (CDN) — 圖示
- **Google Fonts** — Zen Maru Gothic + Noto Sans TC
- **Vanilla JavaScript** — 路由、狀態管理、DOM 操作
- **localStorage** — 本地端持久化（匯率、消費紀錄、清單勾選、自訂項目）

### 附錄：type 類型對照表

| 欄位 | type 值 | 對應圖示 |
|------|---------|---------|
| 交通接送 | `transport` | 🚗 fa-car-side |
| 起飛 | `flight` | ✈️ fa-plane-departure |
| 抵達 | `arrival` | 🛬 fa-plane-arrival |
| 租/還車 | `car` | 🔑 fa-key |
| 餐飲 | `food` | 🍴 fa-utensils |
| 住宿 | `hotel` | 🛏 fa-bed |
| 景點/樂園 | `play` | 🎫 fa-ticket |
| 購物 | `shop` | 🛍 fa-bag-shopping |
| 滑雪 | `person-skiing` | ⛷ fa-person-skiing |

> **提示：** 若有重複天數模式（如連續幾天滑雪），可請 Copilot 建立類似 `skiDay()` 的 helper 函式來簡化。提供地址時請用 **Google Maps 可搜尋的完整地址**，導航功能才能正常運作。
